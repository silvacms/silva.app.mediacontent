======================
silva.app.mediacontent
======================

``silva.app.mediacontent`` a set of simple content types to add and
manage local and remote media content in Silva.

This extension requires Silva 3.x. For previous versions of Silva,
please use an older version of this extension.

Code repository
===============

The code for this extension can be found in Git:
https://github.com/silvacms/silva.app.mediacontent
